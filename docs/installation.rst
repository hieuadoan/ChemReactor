Installation
===========
1. Clone the repository
..code-block:: bash

	git clone https://github.com/hieuadoan/ChemReactor.git
	cd chemreactor

2. Create Conda environment
..code-block:: bash

	conda env create -f chemreactor.yml
	conda activate chemreactor

3. Install in development mode
..code-block:: bash

	pip install -e .
